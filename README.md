## :bulb: Como Utilizar
```bash
  # Clone o repositório:
  $ git clone https://github.com/Matheus-Franco/desafio-vilaApps.git

  # Vá até a pasta:
  $ cd desafio-vilaApps

  # Instale as depências:
  $ yarn install

  # Rode a aplicação:
  $ yarn start
```
